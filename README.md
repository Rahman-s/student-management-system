Student Management System (Django)

This is a web application built using Django framework.

Features:
• User Login Authentication
• Add Student
• Update Student
• Delete Student
• Search Students
• Pagination

Tech Stack:
• Python
• Django
• HTML
• CSS
• SQLite

How to Run Project:

1. Clone repository
2. Install dependencies
3. Run migrations
4. Start server

python manage.py runserver
